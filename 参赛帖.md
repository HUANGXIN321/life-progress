# 人生进度 - 记录你的每一步

## 赛道：生活娱乐

---

## 1. 创意名称 + 创意介绍

**创意名称**：人生进度

**想解决什么问题**：
人们常常忘记自己走过的路、达成的成就，也缺乏一个统一的地方来规划未来目标。生活中那些重要的时刻散落在朋友圈、日记本、备忘录里，没有一个完整的视图来回顾自己的人生轨迹。同时，很多人有目标但缺乏可视化的追踪方式，导致计划容易半途而废。

**为什么会想到做这个**：
灵感来源于"人生进度条"这个概念。当我第一次看到自己已经度过了人生近30%的时光时，那种震撼让我开始认真思考：我做了什么？我想做什么？于是想做一个既能回顾过去、又能规划未来的个人人生管理工具。

**大概是什么产品**：
一个单页网页应用，包含人生进度可视化、时间轴回顾、未来蓝图规划三大核心模块。

---

## 2. 目标用户及痛点

**面向哪些用户**：
- 希望记录人生重要时刻的普通人
- 有目标但缺乏追踪工具的规划者
- 想要回顾成长轨迹的职场人士
- 喜欢可视化数据的生活记录爱好者

**在什么场景下使用**：
- 年末回顾过去一年时
- 设定新年目标时
- 达成某个成就想要记录时
- 感到迷茫想要看看自己已经走了多远时

**当前痛点**：
- 人生重要时刻散落在各处（朋友圈、日记、相册），难以统一回顾
- 目标计划往往只停留在脑子里，缺乏可视化追踪
- 没有直观的"人生进度"概念，容易虚度光阴

---

## 3. 价值与意义

**效率提升**：
将人生记录和目标规划合二为一，减少在多个App之间切换的成本。通过可视化进度条，让抽象的时间变得具体可感。

**社会价值**：
帮助人们更好地认识自己、规划未来。在快节奏的生活中，提供一个停下来审视人生的空间。每一个被记录的时刻都值得被记住，每一个被规划的目标都值得被追求。

---

## 4. 技术实现

**使用工具**：TRAE IDE + AI助手

**技术栈**：
- 纯前端实现：HTML5 + CSS3 + JavaScript
- 响应式设计，适配手机和电脑
- 使用 CSS Grid 和 Flexbox 布局
- Intersection Observer API 实现滚动动画
- 无后端依赖，可部署到任何静态托管平台

**核心功能**：
1. **人生进度总览**：可视化生命进度条 + 统计数据卡片
2. **人生足迹时间轴**：按时间线展示重要事件，支持标签分类
3. **未来蓝图**：展示进行中/计划中的目标，含进度追踪
